# Bingo Quial 50 - MVP

Proyecto Next.js 14 con TailwindCSS para el registro gamificado de familias en el Bingo del Liceo Quial (50 años).

## Uso

```bash
npm install
npm run dev
```

Luego visita `http://localhost:3000/bingo-50`.

Los datos enviados se reciben en `/api/registration` y actualmente solo se escriben en consola. Aquí puedes conectar tu base de datos, Google Sheets, Supabase, etc.
